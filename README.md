# Amaze Go

竖屏箭头消除益智游戏原型。棋盘上的折线按箭头方向逐格前进：每个逻辑帧头部新增一格、尾部删除一格；箭头下一格被占用时扣除一滴水，清除全部折线后过关。

## 运行

```bash
npm install
npm run dev
```

浏览器打开终端显示的本地地址。推荐使用手机竖屏或浏览器移动设备模式。

## 验证

```bash
npm test
npm run build
```
